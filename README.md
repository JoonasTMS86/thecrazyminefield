# thecrazyminefield
The Crazy Minefield - a JavaScript "Minesweeper clone" of mine for 1 to 4 players. Can you cross the minefield without stepping on any of the hidden mines?

Select the number of players (1 to 4) and the number of mines in the options menu. You can also adjust the difficulty of the game,
by setting whether the mine detector in the game can detect mines in all 8 directions or in just 4 directions.

Controls:
Player 1: Arrow Keys to move up, down, left, right
Player 2: W - Up, S - Down, A - Left, D - Right
Player 3: T - Up, G - Down, F - Left, H - Right
Player 4: I - Up, K - Down, J - Left, L - Right
